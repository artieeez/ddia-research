# Obsidian vault

Personal notes synced from [Obsidian](https://obsidian.md/) for this research lab. Point your Obsidian vault at this folder (or symlink it) to keep meeting logs and study notes in the same repo as the code and workshop artifacts.

## Layout

| Folder | Use |
|--------|-----|
| [meetings/](meetings/) | Book club sessions, discussion logs, action items |
| [study/](study/) | DDIA chapter notes, reading highlights, study reflections |

## Conventions

- Use `YYYY-MM-DD` prefixes for dated notes when it helps ordering (e.g. `2026-06-16-ch3-data-models.md`).
- Link to repo docs when relevant — e.g. [next-session.md](../docs/book-club/next-session.md), [state.md](../state.md).
- Curated, polished write-ups can still live under [docs/](../docs/); this vault is for raw capture and personal logs.

## Git

Obsidian config (`.obsidian/`) is ignored so plugin and workspace state stay local. Your markdown notes in `meetings/` and `study/` are tracked normally.
